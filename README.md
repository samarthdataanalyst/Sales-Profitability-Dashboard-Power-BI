# Sales & Profitability Analysis Dashboard

A Power BI dashboard analyzing sales and profitability across 5 countries, 
5 customer segments, and 6 products using Microsoft's Financial Sample dataset.

## Dashboard Preview
![Dashboard Screenshot](Dashboard%20Screenshot.png)

## Key Insights
- Enterprise segment generated $19.6M in sales but posted a net loss of -$614K
- Channel Partners had the highest profit margin (73%) despite lowest revenue
- High-discount transactions averaged 9% margin vs. 18% for low-discount deals
- October and December were the strongest sales months of the year

## Tools Used
- Power BI (Dashboard, visualization)
- Excel (raw data source)
